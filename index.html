most common html attributes

  • id: A unique identifier for an HTML element.
  • class: One or more classes for an HTML element, used for styling or JavaScript.
  • style: Inline CSS styles for an HTML element.
  • src: The URL of an external resource, like an image or script.
  • href: The URL of the linked resource for anchor elements (<a>).
  • alt: Alternative text for images, used when image cannot be displayed.
  • title: Additional information about an element, often displayed as a tooltip.
  • width: The width of an element, like an image or table.
  • height: The height of an element, like an image or table.
  • target: Where to open the linked resource, e.g., _blank for new window/tab.
  • rel: The relationship between the current and linked document for anchor elements (<a>).
  • type: The media type of linked resource for <script> or <style> elements.
  • name: The name of an element, often used with form elements.
  • value: The value of an input element.
  • placeholder: Placeholder text for input elements.
  • disabled: An input element should be disabled.
  • readonly: An input element is read-only.
  • checked: A checkbox or radio button is selected by default.
  • selected: An option in a <select> element is selected by default.
  • autocomplete: Whether an input field should have autocomplete enabled or disabled.

most common css styles

  • width: Width of an element.
  • height: Height of an element.
  • background-color: Background color of an element.
  • color: Text color of an element.
  • font-size: Font size of text.
  • font-family: Font family of text.
  • font-weight: Thickness of text.
  • border: Border of an element.
  • margin: Margin space around an element.
  • padding: Padding space inside an element.
  • display: How an element is displayed (e.g., block, inline, none).
  • position: Positioning method of an element (e.g., static, relative, absolute).
  • float: Floating an element to the left or right.
  • text-align: Alignment of text within an element.
  • text-decoration: Decoration of text (e.g., underline, line-through).
  • box-shadow: Shadow effect around an element.
  • z-index: Stacking order of elements.
